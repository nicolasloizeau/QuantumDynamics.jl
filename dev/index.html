heisenberg(H::Matrix, O::Matrix, observables, dt::Real, tmax::Real)

Heisenberg evolution with dense matrices. Evolve the operator O in the Heisenberg picture under the Hamiltonian H for a time tmax with time step dt. observables is a list of functions, which are applied to the operator O at each time step.
